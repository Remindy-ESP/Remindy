# Remindy ESP

A reminder application built with NestJS backend.

## Structure

- `backend/` - NestJS API server

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
cd backend
npm install
```

### Development

```bash
cd backend
npm run start:dev
```

### Testing

```bash
cd backend
npm test
```

### Build

```bash
cd backend
npm run build
```
